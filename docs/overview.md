# Overview

## What is Zagreb Buddy?

An AI-powered local companion that helps tourists and locals discover the real Zagreb — hidden gems, local events, street art, secret courtyards, and authentic experiences that never appear on tourist maps.

**One-liner:** *Like having a local best friend in Zagreb who always knows the perfect spot.*

## The problem

**Tourists**

- See the same handful of attractions
- Search engines surface tourist traps
- No local context or stories
- Generic itineraries
- Hard to find what is happening *now*

**Locals**

- Forget to explore their own city
- Do not know what is happening nearby
- Hard to find new hidden spots
- Want to share favorites
- No tool built primarily for them

## The solution

A chat-based AI agent that:

- Talks like a local friend, not a search engine
- Knows 30+ hidden gems with real local stories
- Has specialized skills (street art, coffee, architecture, …)
- Considers time, weather, mood, and available hours
- Shows results on an interactive map
- Builds walkable itineraries
- Ships as a mobile-friendly web app (PWA)
